## Install TypeScript And Transpile Files

### install nodejs
```
node -v
```

### install node npm
```
npm -v
```

### install typeScript 
```
npm i -g typeScript
```
### version typeScript
```
tsc -v
```
### compile file
```
tsc index.ts
```
### show hint for command tsc
```
tsc index.ts
```

## Create Configuration And Watch Files
```
tsc -w index.ts
```
## create typeScript confi.json
```
tsc init
```
## compile all file
```
tsc
```
